## Topic 3: Training your first neural network

In this topic, we train a simple neural network using the classic concentric circles classification problem. This dataset contains two classes arranged in nested circular patterns, making it an intuitive example of a nonlinear classification task. The goal is to learn the core workflow for training a model in PyTorch, including preparing data, defining a model, computing loss, optimizing parameters, and evaluating performance.

Please run it on Kaggle by clicking the button below:

[![Open in Kaggle](https://kaggle.com/static/images/open-in-kaggle.svg)](https://www.kaggle.com/kernels/welcome?src=https://github.com/Code-the-Dream-School/python-200/blob/main/lessons/04_ML_deep_learning/resources/notebooks/03_first_neural_network.ipynb)

When you open this notebook in Kaggle, make sure to enable GPU support:

- Start the Kaggle session

- Click Settings in the top menue

- Under Accelerator, select GPU

This will significantly speed up training and is good practice for future deep learning work.
