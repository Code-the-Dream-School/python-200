# Week 8: Introduction to Cloud Computing

Welcome to Week 8 of Python 200 -- and welcome to the cloud. 🌩️

Over the past seven weeks you have built a strong foundation: statistical thinking, classical machine learning, deep learning, large language models, RAG, and AI agents. That is a serious achievement, and you should feel proud! For the remaining four weeks, we will be moving some of your expertise online, to the cloud. 

Cloud platforms are the infrastructure that makes modern data engineering possible -- on-demand compute, scalable storage, managed databases,  ML services, and the pipelines that tie it all together. In the final stretch of this course, you will learn to work in that environment. For the hands-on work, we will be using Microsoft Azure, which is one of the three main cloud platforms in enterprise and nonprofit settings.

This first week will be about getting situated. By the end of it, you will understand the core concepts of cloud computing, have your own workspace set up in Azure, and be ready to hit the ground running in the coming weeks with your environment all set up.

## Topics
1. [Cloud overview](01_cloud_overview.md)
A platform-agnostic introduction to cloud computing: what it is, why it matters, and what changes when you move from running code on your laptop to running it in the cloud. We cover the core concepts you will encounter regardless of which cloud provider you use.

2. [A Practical Introduction to Azure](02_azure_intro.md)
A hands-on introduction to Azure. We will make sure you can log in, navigate the Portal, and get set up with your the sandbox you will use for the rest of the course.
